# 🤖 Naik Kelas Agent Framework

Repositori ini adalah **Root Repository** untuk manajemen instruksi AI, kapabilitas (skills), dan perancangan kurikulum bagi berbagai institusi yang berkolaborasi di platform [naikkelas.id](https://naikkelas.id).

---

## 🏗️ Struktur Proyek
- **`.naikkelas/agents/`**: Berisi panduan instruksi khusus untuk berbagai AI agent (Antigravity, Claude, Gemini, dll.).
- **`skills/`**: Folder untuk menyimpan definisi skill atau kapabilitas AI khusus di masa depan.
- **`koneksi-digital/`** (Git Submodule): Proyek draf kurikulum aktif untuk kolaborasi antara Koneksi Digital dan Yayasan ASIB.

---

## 🤝 Kolaborasi & Ekstensibilitas
Repositori ini dirancang untuk skala besar:
1.  **Multi-Pillar**: Kurikulum di dalam submodule bersifat spesifik (Kreator, Afiliator, Seller).
2.  **Multi-Institution**: Kita dapat menambahkan institusi atau instansi lain sebagai submodule baru di masa depan dengan pola yang sama.
3.  **AI-Native**: Struktur folder `.naikkelas/` memungkinkan berbagai AI agent bekerja secara cerdas dan sinkron di seluruh proyek.
